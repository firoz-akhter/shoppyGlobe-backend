Setup and Installation
    Clone the Repository:
        git clone https://github.com/firoz-akhter/shoppyGlobe-backend.git
        cd shoppyGlobe-backend

    Install Dependencies:
        npm install
    
    Run the Application:
        npm run dev
        Open URL localhost:5173

    Build for Production:
        npm run build
